# Jesse Anderson

I build practical automation, data, and tooling systems that make messy
workflows easier to understand and operate.

- Automation and tooling engineer with a chemical engineering background and an MCS from UIUC
- Focused on C#, Rust, Python, VBA, Excel, embedded systems, and data-heavy workflow tooling
- Interested in internal tools, technical systems, and engineering work that improves clarity and leverage

## Selected Work

- [linreg-core](https://github.com/jesse-anderson/linreg-core)  
  Rust linear regression library published across Rust, WebAssembly, Python, VBA, DLL, and Excel XLL surfaces

- [xll-rs](https://github.com/jesse-anderson/xll-rs)  
  Rust runtime for building Excel XLL add-ins on Windows

- [xllgen](https://docs.rs/xllgen/latest/xllgen/)  
  Proc-macro layer for generating Excel XLL wrappers in Rust

- [xll-utils](https://docs.rs/xll-utils/latest/xll_utils/)  
  PE/COFF parsing and export verification utilities for Excel XLL development

- [Oak Park Crime](https://opcrime.jesse-anderson.net/crime_map)  
  Public-facing crime visualization site and data exploration project for Oak Park

- [N-Back](https://nback.jesse-anderson.net/)  
  Public-facing N-back training site built as a polished end-result project

- [Tools](https://tools.jesse-anderson.net/tools.html)  
  Durable collection of small web utilities published for everyday use

- Workbook Radar  
  Excel/VBA auditing and tracing tool focused on workbook risk, dependency graphs, and VBA flow analysis
